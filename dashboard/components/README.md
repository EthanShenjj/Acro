# Dashboard Components

This directory contains React components for the Acro dashboard.

## Component Structure

- `Sidebar.tsx` - Folder navigation sidebar (Task 24.1)
- `ProjectGrid.tsx` - Grid layout for project cards (Task 25.1)
- `ProjectCard.tsx` - Individual project card component (Task 25.1)
- `EmptyState.tsx` - Empty state UI when no projects exist (Task 25.1)
